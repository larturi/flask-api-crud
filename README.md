# Flask & Postgres CRUD

1. Clonar proyecto

2. Instalar dependencias

```bash
pipenv shell

pipenv install
```

3. Levantar la base de datos Postgres

```bash
docker-compose up -d
```

4. Crear tablas Postgres

```bash
http://127.0.0.1:5000/api/seed
```

5. Ejecutar el servidor

```bash
python app.py
```
